# bmw_aftermarket_prediction

## 📌 Resumen del Proyecto
Este proyecto transforma un desafío técnico crítico en los motores BMW Euro 6 (la saturación de sistemas FAP/EGR) en una **oportunidad estratégica de negocio**. Utilizando analítica predictiva, el modelo identifica vehículos en riesgo antes de que ocurra una avería costosa, permitiendo a la red de posventa actuar de forma proactiva.

### 🎯 Objetivos de Negocio
* **Customer Care:** Eliminar la frustración del cliente ante averías críticas inesperadas.
* **Rentabilidad:** Convertir reparaciones reactivas de alto coste (2.500€) en servicios preventivos de alto margen (350€).
* **Fidelización:** Aumentar el *Customer Lifetime Value* (CLV) mediante el cuidado proactivo del vehículo.

## 💻 Código y Metodología
El proyecto incluye un script de procesamiento de datos que clasifica las unidades según su nivel de saturación y asigna prioridades de intervención basadas en telemetría sintética de la DDE (Digital Diesel Electronics).

### Lógica de Priorización:
- **CRÍTICO:** > 90% Saturación (Sustitución inminente).
- **PREVENTIVO:** 70% - 85% Saturación (Punto óptimo de intervención).
- **SALUDABLE:** < 50% Saturación (Ciclos de regeneración normales).

---

## 📈 Impacto en el P&L
Gracias a esta solución, una concesión puede optimizar su **Workshop Absorption Rate** al programar intervenciones de baja complejidad y alta rotación, mejorando el flujo de caja y liberando capacidad para reparaciones mayores.
