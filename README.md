# bmw_aftermarket_prediction
Predictive Maintenance &amp; Aftermarket Strategy: BMW Diesel Emission Systems (Euro 5/6)

Problema: Los sistemas de tratamiento de gases son el principal motivo de entrada a taller en motores diésel modernos debido a ciclos de uso inadecuados (exceso de ciudad).

Solución: Un modelo de datos que identifica vehículos en riesgo de avería inminente para transformar una reparación correctiva costosa en un mantenimiento preventivo rentable.
